# React Engineering Playbook

Welcome to the React Engineering Playbook.

This playbook provides the standard workflow followed by CDIS engineers while developing React applications.

> **Note**
>
> This guide does **not** teach React.
> It assumes you already know React and focuses on how React applications are built, configured, and maintained at CDIS.

---

## Who Should Use This?

- New engineers joining CDIS
- Engineers starting a new React project
- Engineers contributing to an existing React project

## General Rules

While working on React projects at CDIS:

- Follow the guides in order.
- Follow the standard folder structure.
- Follow coding standards.
- Follow naming conventions.
- Use only approved project configurations.
- Open Pull Requests for all code changes.
